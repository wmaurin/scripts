# Scripts

A collection of utility scripts for development and system management.

## Bash Scripts

- **[cleandocker.sh](bash/cleandocker.sh)** - Clean up Docker containers, images, and volumes
- **[dotfiles.sh](bash/dotfiles.sh)** - Manage dotfiles repository setup and synchronization
- **[keygen.sh](bash/keygen.sh)** - Generate SSH keys for development
- **[packages.sh](bash/packages.sh)** - Update system packages and install development tools

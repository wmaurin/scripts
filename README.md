# Scripts

A collection of utility scripts for development and system management.

## Shell Scripts

- **[shell/cleandocker.sh](shell/cleandocker.sh)** - Clean up Docker containers, images, and volumes
- **[shell/dotfiles.sh](shell/dotfiles.sh)** - Manage dotfiles repository setup and synchronization
- **[shell/keygen.sh](shell/keygen.sh)** - Generate SSH keys for development
- **[shell/packages.sh](shell/packages.sh)** - Update system packages and install development tools
